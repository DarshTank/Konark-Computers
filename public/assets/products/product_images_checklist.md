# TechLab Product Images Checklist

The application logic has been updated to use specific local images for each product. 
Since I cannot download these files directly, please download relevant images for the following products and place them in `public/assets/products/` with the exact filenames listed below.

## Process
1.  Use Google Images to search for the Product Name.
2.  Download a clean image (box art or product shot).
3.  Rename the file to match the **Required Filename**.
4.  Move it to `d:\JavaScript\konark-computers\public\assets\products\`.
5.  Go to `http://localhost:3000/admin/settings` (or wherever the seeder is) and run the **Database Seeder**.
